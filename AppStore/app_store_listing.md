## 二、应用描述（中文）

### 简短描述（30 字以内，用于搜索结果）

直连 SQLite / MySQL，随时随地管理你的数据库。

### 完整描述（4000 字符以内）

```
THDatabase 是一款专为开发者和数据库管理员设计的原生 iOS 数据库客户端。
无需后端服务器，直接从 iPhone 或 iPad 连接并操作你的数据库。

【支持的数据库类型】
• SQLite — 管理设备本地 .db / .sqlite 文件，完整 CRUD 操作
• MySQL — 通过 TCP 连接远程 MySQL 服务器，执行任意 SQL 查询

【核心功能】
• 多连接管理：保存、编辑、删除数据库连接配置
• SQL 编辑器：输入并执行任意 SQL 语句
• 结果浏览：以表格形式展示查询结果，支持横向滚动
• Schema 浏览：侧边栏快速查看数据库列表与表结构
• 写操作保护：执行 INSERT / UPDATE / DELETE / DROP 等高危操作前弹出确认对话框
• 安全存储：数据库密码通过 iOS Keychain 加密保存，不保存明文

【为什么选择 THDatabase】
• 纯原生 UIKit 实现，流畅、轻量
• 无后端依赖，数据不经过任何第三方服务器
• 密码存储采用 iOS Keychain，安全可靠
• 适合开发调试、运维巡检、数据核查等场景

【技术特性】
• 基于 MySQLNIO 的高性能 MySQL 连接
• 系统原生 SQLite 框架支持
• 统一的数据库适配器架构
• 完整的错误处理与用户提示

有任何建议或问题，欢迎通过 GitHub Issues 联系我们。
```

## 三、应用描述（英文）

### Subtitle

Lightweight database client for iOS

### Full Description

```
THDatabase is a native iOS database client designed for developers and DBAs.
Connect directly to your databases from iPhone or iPad — no backend server required.

【Supported Databases】
• SQLite — manage local .db / .sqlite files on device with full CRUD operations
• MySQL — connect to remote MySQL servers over TCP and execute any SQL query

【Key Features】
• Multi-connection management: save, edit, and delete connection profiles
• SQL editor: write and execute any SQL statement
• Result viewer: display query results in a scrollable table view
• Schema browser: sidebar for quick navigation of databases and tables
• Write protection: confirmation dialog before executing INSERT / UPDATE / DELETE / DROP
• Secure storage: passwords encrypted in iOS Keychain, never stored in plain text

【Why THDatabase】
• Pure native UIKit — fast and lightweight
• No third-party servers involved; your data stays private
• iOS Keychain for password storage
• Great for development debugging, ops checks, and data inspection

【Technical Features】
• High-performance MySQL connections via MySQLNIO
• Native iOS SQLite framework support
• Unified database adapter architecture
• Comprehensive error handling and user feedback

Feedback and suggestions are welcome via GitHub Issues.
```

## 十四、审核注意事项（App Review Notes）

在 App Store Connect 提交页面的「App Review Information」→「Notes」填写：

```
Dear App Review Team,

THDatabase is a native iOS database client for developers.

Testing Instructions:
1. Launch the app — you will see the connection list screen.
2. Tap "+" to add a new SQLite connection:
   - Select "SQLite" as the database type
   - Enter any display name (e.g., "Test DB")
   - Leave the host field blank (SQLite uses local files)
   - Tap "Save"
3. Tap the saved connection to open the database workspace.
4. Enter a SQL query in the editor, e.g.: SELECT sqlite_version();
5. Tap "Run" to execute and view results in the table below.

No external server is required for SQLite testing.
For MySQL testing, a live MySQL server would be needed (not provided for review).

Security note:
- The app uses iOS Keychain only for storing database passwords locally.
- No data is transmitted to any third-party server.
- All database connections are made directly from the device.

Thank you for your review.
```

## 十六、GitHub Pages 部署步骤

为了让隐私政策和服务条款可公开访问：

1. 进入 GitHub 仓库：`https://github.com/leotan1990/databases`

4. 选择分支：**main** 和目录：**/ (root)**
